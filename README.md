# ThreeInRowGame
ThreeInRowGame — это настольная игра-головоломка, реализованная на Java с использованием библиотеки Swing. Игроку необходимо собирать линии из трёх (или более) элементов одного цвета для их удаления и получения очков.

## Особенности игры
- Игровое поле: сетка размером 8x8 клеток.
- Элементы: 5 типов элементов (каждый цвет представляет определённый тип).
- Счёт: за каждую удалённую клетку игрок получает 10 очков.
- Игровой процесс:
  1. Игрок выбирает две соседние клетки, чтобы поменять их местами.
  2. Если после перемещения образуется линия из трёх (или более) элементов одного цвета по горизонтали или вертикали, линия удаляется.
  3. На освободившиеся места сверху опускаются (и добавляются) новые элементы, а игрок получает очки: по 10 за каждый удалённый элемент.
  4. После каждого удаления проверяется, образуются ли новые совпадения, и они также удаляются с добавлением очков. Пустые клетки заполняются новыми элементами.
  5. Если совпадений нет, обмен отменяется.

## Запуск игры
### Требования
- Java Development Kit (JDK) версии 8 или выше;
- Среда разработки IntelliJ IDEA или любой другой текстовый редактор для работы с Java.

### Инструкция по запуску
1. Клонируйте репозиторий:
```bash
git clone https://github.com/ваш_пользователь/ThreeInRowGame.git
2. Откройте проект в IntelliJ IDEA.
3. Убедитесь, что файл ThreeInRowGameSave.java находится в папке src/ru/varvara.
4. Скомпилируйте и запустите проект:
  - В IntelliJ IDEA щёлкните правой кнопкой мыши на ThreeInRowGameSave.java и выберите Run 'ThreeInRowGameSave.main()'.
5. Игровое окно откроется, и вы сможете начать играть.

### Управление
1. Кликните по первой клетке, чтобы выбрать её.
2. Кликните по соседней клетке, чтобы поменять их местами.
3. Если образуется линия из трёх или более одинаковых элементов, они удалятся, а вы получите очки.
